# 🪨 Rock Paper Scissors – Python Game

A simple terminal-based **Rock, Paper, Scissors** game where you play against the computer. The game is written in Python and is beginner-friendly, using only built-in modules.

## 🎮 How to Play

- Choose between **rock**, **paper**, or **scissors**.
- The computer randomly selects its choice.
- The winner is decided based on classic rules:
  - Rock beats Scissors
  - Scissors beats Paper
  - Paper beats Rock
- The game continues until you decide to stop.

## ✅ Features

- Simple and clean user input handling
- Randomized computer choice using Python’s `random` module
- Loop for replaying without restarting the program
- Friendly terminal output

## 🧱 Requirements

- Python 3.x
- No external libraries required
